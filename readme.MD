# Landleren.be

This repo contains the code for a website students of [Thomas More applied IT](https://thomasmore.be/nl/opleidingen/professionele-bachelor/toegepaste-informatica) made for "De Regenboog", a local primary school. It's a website that helps you memorize the countries and flags of Europe.
